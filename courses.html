HTML and JavaScript

  Monday-Friday, 6/21 to 7/2/21          
  5 - 5:45pm PST

  Adrian Baker

This course is a JavaScript (along with a bit of HTML) programming language introduction geared towards middle and high school students. No prior programming experience is necessary. Concepts covered include variables, mathematical and logical operators, control flow, loops, functions, JavaScript web APIs, arrays, and more! This course will emphasize self-direction, problem-solving, and hands-on practice—be prepared to google things, write lots of code, and most importantly, think! Students are also encouraged to go as far ahead of the class schedule as they want and explore beyond class content if they can.

JavaScript itself is a programming language widely used across many fields. It is used most extensively in frontend web development, effectively the gold standard for that purpose. Nearly every web page you visit uses JavaScript somewhere. JavaScript is present in the backend with programs such as Node.js and desktop application development with Electron. It is also a scripting language for many software programs, including but not limited to Adobe After Effects and MuseScore. This course focuses somewhat on the frontend development aspects of JavaScript, though the skills learned here apply to all areas and even to different languages entirely.

Important Disclaimer: JavaScript is not Java. “JavaScript” is to “Java” as “carpet” is to “car.” The two are entirely different languages with no relation!
